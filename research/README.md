# Research

Notas de investigación en bruto: capturas directas de ejecutar un prompt de `prompts/`
contra una fuente, sin depurar ni validar todavía.

Esta carpeta es la capa de **trabajo en progreso**, distinta de `knowledge/`, que es la
capa curada y validada. Nada de lo que hay acá debe considerarse verdad establecida hasta
que pase por el proceso descrito en `docs/research/WORKFLOW.md`.

## Convención
- Una nota por sesión de investigación, usando `templates/research.md`.
- Nombre de archivo: `YYYY-MM-DD-tema-en-kebab-case.md`.
- Cuando una nota se destila a `knowledge/`, se deja en `research/` como referencia
  histórica (no se borra), con un enlace a la entrada final.
