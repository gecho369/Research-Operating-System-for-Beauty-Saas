# Estándar de escritura

Cómo se escribe cualquier documento dentro de este repositorio.

## Reglas
1. **Distinguir siempre**: hecho, inferencia, hipótesis u opinión. No mezclar sin etiquetar.
2. **Una idea por párrafo.** Frases cortas, directas.
3. **Cero relleno.** Sin introducciones genéricas ni conclusiones redundantes.
4. **Citar la fuente** de cada afirmación no obvia.
5. **Indicar incertidumbre explícitamente** cuando exista, y qué la reduciría.
6. **Escribir para quien no investigó esto.** El contexto necesario va en el documento,
   no se asume.

## Estructura sugerida para una entrada de conocimiento
1. Conclusión / hallazgo principal.
2. Evidencia que lo sostiene.
3. Limitaciones o riesgos del hallazgo.
4. Implicancia para el producto (si la hay).
