# Principios operativos

Reglas que gobiernan cómo se investiga y cómo se documenta el conocimiento en este sistema.

1. **Evidencia antes que opinión.** Toda afirmación relevante debe poder trazarse a una fuente.
2. **Separar hechos de inferencias.** Un hallazgo y la conclusión que se saca de él no son lo mismo.
3. **Nada se pierde, se archiva.** El conocimiento superado va a `archive/`, no se borra.
4. **Un solo lugar de verdad.** `knowledge/` es la fuente autorizada; `research/` es material de trabajo.
5. **Todo documento tiene dueño y fecha.** Sin autoría ni fecha, un documento no es confiable.
6. **Escribir para quien no estuvo en la investigación original**, siguiendo `WRITING_STANDARD.md`.

_(Ampliar o ajustar según el criterio del equipo)_
